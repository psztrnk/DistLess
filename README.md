# DistLess

Really simple PHP distractionless writer.

**Anti-features:**
- Set `$scheme` to `'dark'` in Line 3 of `index.php` for dark color scheme!
- Set `$scheme` to `'light'` in Line 3 of `index.php` for light color scheme!
- Set filename!
- Choose TXT or Markdown format!
- Write!
- Save your file!
